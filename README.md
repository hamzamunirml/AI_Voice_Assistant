##  Project Overview
This project is a Python-based Voice Assistant that allows users to control their computer using voice commands. It utilizes Speech Recognition and Text-to-Speech technologies to interact with users and perform basic system-level operations.

The assistant listens to user commands through a microphone, processes them, and executes corresponding tasks such as opening applications, searching the web, controlling system volume, and more.

---

##  Objective
To develop a simple and user-friendly voice-controlled assistant capable of performing basic computer operations and improving human-computer interaction.

---

##  System Working
The system follows a simple pipeline:

1. Capture voice input via microphone  
2. Convert speech to text using SpeechRecognition  
3. Process the command using predefined logic  
4. Execute the corresponding task  
5. Respond using Text-to-Speech  

---

## ⚙️ Features

-  Voice Command Recognition  
-  Text-to-Speech Feedback  
-  Open Websites (Google, YouTube)  
-  Open Applications (Notepad, VS Code)  
-  Google Search via Voice  
-  Play Music from YouTube  
-  Volume Control (Increase / Decrease / Mute)  
-  Get Current Time  
-  System Control (Shutdown / Restart)  
-  Exit Command Support
